# CeylonEcoTrails – Eco Tourism Booking Platform

Overview:

Academic full-stack web development project implementing an eco-tourism booking platform. The system allows users to browse tours, manage bookings through a session-based cart, and complete payments using PayHere Sandbox integration.

Features:
1. Tour browsing system
2. Session-based booking cart
3. Add / update / remove bookings
4. Multi-page responsive website
5. Database-driven booking system
6. PayHere Sandbox payment gateway integration

Technologies Used:
1. HTML, CSS
2. JavaScript
3. PHP
4. MySQL (via phpMyAdmin in XAMPP)

Database:
1. Includes SQL file: `trekking (1).sql`

Project Structure
1.  `index.php` → Home page  
2. `tours.php` → Tour listing  
3. `booking.php` → Booking form  
4. `cart.php` → Cart + payment integration  
5. `contactUs.php` → Contact page  
6. `db.php` → Database connection  
7. `script.js` → Frontend interactions  
8. `cart.css` → Styling  

Note:

This project was developed as part of an academic coursework assignment focused on full-stack web development and database-driven applications.
